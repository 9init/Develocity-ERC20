# ERC-20-task

A quick ERC-20 task

## Prerequisites
- Node v18.12.1

## Start
1 - Install the dependencies using your favorite package manager, I'm using yarn.
- for yarn
```bash
$ yarn install
```
- for npm
```bash
$ npm install
```

2 - Run the server
```bash
$ node app
```
